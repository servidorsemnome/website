**Installing**

Add this line to your `/etc/hosts` (Linux) or `%WinDir%\System32\Drivers\Etc\hosts` (Windows) file:

```sh
127.0.0.1 local.ssn.gg
```

```sh
npm install
# or with yarn
yarn
```

**Running**

```sh
npm run dev
# or with yarn
yarn dev
```

Open [local.ssn.gg:3000](http://local.ssn.gg:3000) on your browser of choice.
